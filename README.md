# Typola
Typing practice with instant feedback to learn a language.

## Development Setup

1. Clone the repository:
   `git clone https://github.com/lu7ue/cappy.git`
2. Navigate to the root:
    `cd cappy`
3. Install backend/Electron dependencies:
   `npm install`
4. Install frontend dependencies:\
   go to the frontend directory, `cd src/frontend`\
   then, `npm install`
5. Start the app:\
   back to the project root directory, `cd ../..`\
   then, `npm run dev`

The Electron window and Vite dev server will start automatically.
